# Release Notes

## v0.1.2 — UI/UX改善

試用時に感じたUI上の違和感を解消するための改善リリース。

### 変更

- **吹き出しをコンパクト化**: 発言者ラベルと投稿時間を同じ行に統合し、1行本文でも縦に間延びしない2行構成にした
- **「保存」に表記変更**: ヘッダーの「上書き保存」を「保存」に変更
- **「終了処理」ボタンを追加**: 書き終えた後の操作（Markdownコピー・IdeaNest用コピー・全件削除）を「終了処理」ダイアログに集約
- **コピー時に自動保存**: 終了処理からコピーを選ぶと保存してからコピーを実行。保存先未設定の場合はダイアログを表示し、キャンセルするとコピーも中止される
- **「名前を付けて保存」ボタンを追加**: ヘッダーに常時表示

### 注意

v0.1.x はプロトタイプです。今後正式採用するかは利用感を見て判断します。

## v0.1.1 — 起動導線改善

ChatNest を日常的に試用しやすくするための起動導線改善リリース。

### 追加・変更

- **スタートダイアログ**: 通常起動時にダイアログを表示し、新規チャットまたは最近開いたファイルを選択して開始できる
- **最近開いたファイル**: 開いた `.chatnest` ファイルを最大5件記録（`%APPDATA%\ChatNest\settings.json`）
- **ファイル関連付け起動**: `.chatnest` ファイルを EXE に手動関連付けした場合、ダブルクリックでそのファイルを読み込んで起動できる
- **新規ボタン**: ヘッダーに「新規」ボタンを追加
- **ウィンドウタイトル**: 開いているファイル名をタイトルバーに表示

### 注意

v0.1.x はプロトタイプです。今後正式採用するかは利用感を見て判断します。

## v0.1.0 — 初期プロトタイプ

ChatNest の最初のプロトタイプリリース。

チャット形式で思考を一時的に吐き出し、Markdown / IdeaNest 形式でコピーする感覚を確認するための試作品です。

### 機能

- 発言者（自分 / 反論 / 補足 / 要約）を切り替えてチャット形式で投稿
- `Ctrl + Enter` で投稿
- 発言の個別削除・全件削除（確認ダイアログあり）
- Markdown 形式で全件コピー
- IdeaNest 貼り付け用テキストとして全件コピー
- `.chatnest` ファイルへの保存・読み込み

### 注意

v0.1.0 はプロトタイプです。今後正式採用するかは利用感を見て判断します。
