## v0.4.0 — AppShell / WorkspaceView 責務分離

内部構造の整理リリース。UI/UX・ファイル形式・操作感に変更なし。

### 変更

- **`ChatNestWorkspaceView` / `ChatNestWorkspaceViewModel` を新設**: メッセージ一覧・入力欄・発言者選択・投稿・削除・発言者ショートカット・コピーテキスト生成をUserControlとして切り出し
- **`MainWindow` / `MainViewModel` をアプリ外枠として整理**: ファイル操作・スタートダイアログ・終了処理・ウィンドウタイトル・最前面トグルを担当。ワークスペース操作はWorkspaceへ委譲
- **最前面トグルをヘッダーへ移動**: 発言者選択欄の右端からヘッダーのボタン列に移動（機能は同一）
- **`WorkspaceModified` イベントパターン**: WorkspaceViewModelが変更イベントを発行 → MainViewModelが購読 → 変更時保存を実行

コピー出力の読みやすさを改善する「出口改善」リリース。ChatNest本体の機能追加はなし。

## v0.3.0 — 保存安全性・メニュー整理

日常利用時のメッセージ保護と操作整理のリリース。

### 変更

- **変更時保存**: 保存済みファイルを開いている状態でメッセージ送信・削除すると、現在ファイルへ即時保存する。未保存（新規状態）では送信・削除時に保存ダイアログを出さない
- **ヘッダーメニュー整理**: 「新規」ボタンを削除。「保存」→「上書き保存」、「名前を付けて保存」→「新規保存」に改称
- **「全件削除」を廃止**: 変更時保存の導入により、全件削除後の空状態が意図せず保存されるリスクを排除するため廃止
- **終了処理に「新規チャットを開始」を追加**: 空の状態から始めたい場合は、現在ファイルを空にして保存する操作ではなく、ファイルを閉じて新規状態にする操作として実装

### 方針

変更時保存はタイマー型の自動保存ではなく、操作（送信・削除）に連動した「変更時保存」とする。

## v0.2.0 — 正式採用に向けた基礎整備

v0.1.x 試用の結果、正式採用方針に移行。軽量思考ログツールとしての方針を固め、基本動作を安定化するリリース。

### 変更

- **正式採用方針に移行**: v0.1.x はプロトタイプ前提だったが、v0.2.0 から正式採用前提に切り替え
- **READMEを正式採用前提に書き直し**: ChatNest / IdeaNest / NoteNest の役割分担、軽さを守る基本方針を明文化
- **未保存変更の確認**: 終了・新規作成・別ファイルを開く際に未保存の変更があれば「保存する / 保存しない / キャンセル」の確認を表示
- **保存・読込の安定化**: ファイル操作まわりの基本動作を確認・整備

### 方針

ChatNestは軽さを保つことを重視します。複数チャット管理・分類・検索・蓄積が必要になった場合は、ChatNestに機能を追加するのではなく、IdeaNestへ移すことを基本方針とします。

## v0.1.3 — 入力体験・表示密度改善

試用時に感じた入力操作と表示密度の違和感を解消するリリース。

### 変更

- **タイトルバーにバージョン表示**: ファイルなし → `ChatNest - ver0.1.3`、ファイルあり → `ChatNest - ファイル名 - ver0.1.3`
- **「要約」を「結論」に改名**: 発言者の選択肢を `自分 / 反論 / 補足 / 結論` に変更。旧データの `要約` は読み込み時に `結論` へ自動変換
- **Shift+Enter で投稿**: Ctrl+Enter に加えて Shift+Enter でも投稿可能に
- **発言者切替ショートカット**: Ctrl+→/← および Shift+→/← で発言者を順次切替。入力中の本文は消えない
- **発言者・時刻を吹き出し外に移動**: メタ情報（発言者・時刻・✕）を吹き出しの左に横並びで配置し、吹き出し内は本文のみに。1行本文のメッセージ高さを約36pxに削減（v0.1.2比 約18%削減）

### 注意

v0.1.x はプロトタイプです。今後正式採用するかは利用感を見て判断します。

## v0.1.1 — 起動導線改善

### 追加・変更

### 機能

- 発言者（自分 / 反論 / 補足 / 要約）を切り替えてチャット形式で投稿

- 発言の個別削除・全件削除（確認ダイアログあり）
- Markdown 形式で全件コピー
- IdeaNest 貼り付け用テキストとして全件コピー
- `.chatnest` ファイルへの保存・読み込み

### 注意

v0.1.0 はプロトタイプです。今後正式採用するかは利用感を見て判断します。
