# ChatNest

**バージョン: v0.1.3（プロトタイプ）**

---

## ChatNest とは

ChatNest は、まだ IdeaNest のカードにも NoteNest のノートにもできない考えを、チャット形式で一時的に吐き出すためのローカルデスクトップツールです。

LINE のようなチャット UI を一人で使うイメージで、「自分・反論・補足・結論」という発言者を切り替えながら自問自答・ブレストを行います。  
ある程度まとまったら、Markdown 形式でコピーして IdeaNest や NoteNest に貼り付けます。

---

## v0.1.x はプロトタイプ

v0.1.x は正式採用前の感覚確認用試作品です。  
「チャット形式で思考を吐き出す感覚がよいか」を確認することを目的としており、機能は最小限に絞っています。

今後正式採用するかどうかは、実際の利用感を見て判断します。

---

## IdeaNest / NoteNest との役割分担

| ツール | 役割 |
|--------|------|
| **ChatNest** | 考えをチャット形式で一時的に吐き出す場所 |
| **IdeaNest** | アイデアをカードとして蓄積・整理する場所 |
| **NoteNest** | プロジェクトのノート・タスクを管理する場所 |

ChatNest は IdeaNest より前段の一時作業場です。  
まとまったら ChatNest からコピーし、IdeaNest / NoteNest に貼り付けて使います。

## 使い方

### 起動

- **通常起動**: アプリを起動するとスタートダイアログが表示されます。  
  「新規チャットを開始する」または最近開いたファイルを選択して開始します。

- **ファイルから起動**: `.chatnest` ファイルを ChatNest.exe に手動で関連付けすると、  
  ファイルをダブルクリックした際にそのファイルを読み込んで起動します。  
  ※ファイル関連付けは Windows の「プログラムから開く」で手動設定してください。

### 投稿

1. 画面下部の発言者（**自分 / 反論 / 補足 / 結論**）を選択する
2. テキストエリアに考えを書く
3. **Ctrl + Enter** または **Shift + Enter** または「投稿」ボタンで投稿する

### キーボードショートカット

| ショートカット | 動作 |
|--------------|------|
| `Ctrl + Enter` / `Shift + Enter` | 投稿 |
| `Enter` | 改行 |
| `Ctrl + →` / `Shift + →` | 次の発言者へ（自分→反論→補足→結論→自分） |
| `Ctrl + ←` / `Shift + ←` | 前の発言者へ |

発言者切替ショートカット使用時、入力中の本文は消えません。

### コピー機能

| ボタン | 内容 |
|--------|------|
| **Markdownコピー** | `## 発言者` 形式のMarkdownとしてコピー |
| **IdeaNest用コピー** | `【発言者】` 形式でコピー |

### ファイル操作

| ボタン | 内容 |
|--------|------|
| **新規** | 新規チャットを開始（確認ダイアログあり） |
| **開く** | `.chatnest` ファイルを開く |
| **保存** | 現在のファイルに保存（未保存なら名前を付けて保存） |
| **名前を付けて保存** | 新しいファイルとして保存 |

### 終了処理

書き終えたら「**終了処理**」ボタンを押します。以下の操作を選択できます。

| 操作 | 内容 |
|------|------|
| **Markdownコピー** | 保存してから `## 発言者` 形式でコピー |
| **IdeaNest用コピー** | 保存してから `【発言者】` 形式でコピー |
| **全件削除** | ログをすべて削除（確認あり） |

コピー操作は、保存を先に行います。保存先が未設定の場合はダイアログが表示され、キャンセルするとコピーも中止されます。

### その他の操作

- 各発言の **✕ボタン**: その発言を個別に削除（確認ダイアログあり）

## v0.1.x 実装対象外

以下は意図的に実装していません。

- AI 応答 / 外部 API 連携 / クラウド連携
- タグ / 検索 / ピン留め / アーカイブ
- タスク管理 / ノート管理 / カード管理
- 自動保存 / 複数セッション管理
- IdeaNest / NoteNest ファイルへの直接書き込み
- ファイル関連付けの自動登録 / インストーラー作成

「情報を管理する」ツールではなく、「一時的に吐き出してコピーするツール」として設計しています。

## 動作環境

- Windows 10 / 11
- .NET 8 ランタイム

## ビルド方法

```bash
cd ChatNest
dotnet build ChatNest.sln
dotnet run --project ChatNest/ChatNest.csproj
```
