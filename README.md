# ChatNest

**バージョン: v0.3.0**

---

## ChatNestとは

ChatNestは、気軽に思考を流し込む、チャット式の軽量思考ログツールです。

LINE のようなチャット UI を使いながら、発言者（**自分・反論・補足・結論**）を切り替えて自問自答やブレストを行います。  
考えがまとまったら、Markdown形式でコピーして IdeaNest や NoteNest に移します。

---

## Nest系ツールの役割分担

| ツール | 役割 |
|--------|------|
| **ChatNest** | 気軽に思考を流す、一時的なチャット式思考ログ |
| **IdeaNest** | 残したいアイデアをカードとして蓄積・見返す |
| **NoteNest** | プロジェクトとしてノート・タスクを整理する |

ChatNestは IdeaNest の前段にある一時作業場です。  
考えがまとまったら ChatNest からコピーし、IdeaNest や NoteNest に貼り付けて使います。

---

## 基本方針

ChatNestは、**軽さを保つことを重視します**。

複数チャット管理・分類・検索・蓄積が必要になった場合は、ChatNestに機能を追加するのではなく、**IdeaNestへ移すことを基本方針**とします。

## 使い方

### 起動

- **通常起動**: スタートダイアログが表示されます。「新規チャットを開始する」または最近開いたファイルを選択して開始します。
- **ファイルから起動**: `.chatnest` ファイルを ChatNest.exe に手動で関連付けると、ダブルクリックでそのファイルを読み込んで起動します。  
  ※ファイル関連付けは Windows の「プログラムから開く」で手動設定してください。

### 投稿

1. 発言者（**自分 / 反論 / 補足 / 結論**）を選択する
2. テキストエリアに考えを書く
3. **Ctrl + Enter** または **Shift + Enter** で投稿する

### キーボードショートカット

| ショートカット | 動作 |
|--------------|------|
| `Ctrl + Enter` / `Shift + Enter` | 投稿 |
| `Enter` | 改行 |
| `Ctrl + →` / `Shift + →` | 次の発言者へ（自分→反論→補足→結論→自分） |
| `Ctrl + ←` / `Shift + ←` | 前の発言者へ |

発言者切替ショートカットを使用しても、入力中の本文は消えません。  
発言者選択欄の右側に常時ヒントを表示しています（`Ctrl / Shift + ← / → で切り替え`）。

### 変更時保存

保存済みファイルを開いている状態では、メッセージ送信・削除のたびに現在ファイルへ自動保存します。  
まだ保存先がない新規状態では、送信・削除時に保存ダイアログは表示されません（手動保存または終了処理時の確認に委ねます）。

### ファイル操作

| ボタン | 動作 |
|--------|------|
| **開く** | `.chatnest` ファイルを開く（未保存の変更がある場合は確認あり） |
| **上書き保存** | 現在のファイルに保存（保存先がない場合はダイアログを表示） |
| **新規保存** | 新しいファイルとして保存 |

未保存の変更がある状態でウィンドウを閉じようとした場合も、保存確認が表示されます。

確認ダイアログの選択肢：
- **保存する** — 保存後に操作を続行
- **保存しない** — 保存せず操作を続行
- **キャンセル** — 元の画面に戻る

### 最前面表示

発言者選択欄の右端にある「**最前面**」ボタンで、ウィンドウを常に最前面に固定できます。

| 状態 | 動作 |
|------|------|
| OFF（初期） | 通常のウィンドウ表示 |
| ON | 他ウィンドウより前面に常時表示 |

他の作業をしながら入力したいときに ON にし、煩わしくなったら OFF または最小化してください。

### コピー機能

| ボタン | 内容 |
|--------|------|
| **Markdownコピー** | `## 発言者` 形式のMarkdownとしてコピー |
| **IdeaNest用コピー** | `【発言者】` 形式でコピー |

どちらのコピーも、連続する同じ発言者カテゴリのメッセージを集約して出力します。  
途中で別カテゴリを挟んだ場合は、同じ発言者でも再度カテゴリ名が出力されます。  
メッセージ本文の要約・編集は行いません。

### 終了処理

「**終了処理**」ボタンから以下の操作を選択できます。

| 操作 | 内容 |
|------|------|
| **Markdownコピー** | 保存してから `## 発言者` 形式でコピー |
| **IdeaNest用コピー** | 保存してから `【発言者】` 形式でコピー |
| **新規チャットを開始** | 現在のファイルを閉じて空の新規状態にする（現在ファイルの内容は変えない） |

コピー操作は保存を先に行います。保存先が未設定の場合はダイアログが表示され、キャンセルするとコピーも中止されます。

空の状態から始めたい場合は「新規チャットを開始」を使用してください。現在開いているファイルはそのまま残り、新しい空の作業場が開きます。

### その他の操作

- 各発言の **✕ボタン**: その発言を個別に削除（確認ダイアログあり）

---

## 実装しない機能

ChatNestは「気軽に入力してメッセージとして並べる」ツールです。  
以下は意図的に実装しません。

- 複数チャット管理 / フォルダ分類 / タグ / 検索 / ピン留め / アーカイブ
- AI応答 / 外部API連携 / クラウド連携
- タスク管理 / カード管理
- 自動保存 / 複数セッション管理
- IdeaNest / NoteNest ファイルへの直接書き込み
- ファイル関連付けの自動登録 / インストーラー作成

複数チャットの管理や蓄積が必要になった場合は、IdeaNestへ移してください。

## 動作環境

- Windows 10 / 11
- .NET 8 ランタイム

## ビルド方法

```bash
cd ChatNest
dotnet build ChatNest.sln
dotnet run --project ChatNest/ChatNest.csproj
```
