# ChatNest

**バージョン: v0.1.0（プロトタイプ）**

---

## ChatNest とは

ChatNest は、まだ IdeaNest のカードにも NoteNest のノートにもできない考えを、チャット形式で一時的に吐き出すためのローカルデスクトップツールです。

LINE のようなチャット UI を一人で使うイメージで、「自分・反論・補足・要約」という発言者を切り替えながら自問自答・ブレストを行います。  
ある程度まとまったら、Markdown 形式でコピーして IdeaNest や NoteNest に貼り付けます。

---

## v0.1.0 はプロトタイプ

v0.1.0 は正式採用前の感覚確認用試作品です。  
「チャット形式で思考を吐き出す感覚がよいか」を確認することを目的としており、機能は最小限に絞っています。

今後正式採用するかどうかは、実際の利用感を見て判断します。

---

## IdeaNest / NoteNest との役割分担

| ツール | 役割 |
|--------|------|
| **ChatNest** | 考えをチャット形式で一時的に吐き出す場所 |
| **IdeaNest** | アイデアをカードとして蓄積・整理する場所 |
| **NoteNest** | プロジェクトのノート・タスクを管理する場所 |

ChatNest は IdeaNest より前段の一時作業場です。  
まとまったら ChatNest からコピーし、IdeaNest / NoteNest に貼り付けて使います。

---

## 使い方

1. アプリを起動する
2. 画面下部の発言者（**自分 / 反論 / 補足 / 要約**）を選択する
3. テキストエリアに考えを書く
4. **Ctrl + Enter** または「投稿」ボタンで投稿する
5. ログが積み上がったら、ヘッダーのボタンでコピーする

### コピー機能

| ボタン | 内容 |
|--------|------|
| **Markdownコピー** | `## 発言者` 形式のMarkdownとしてコピー |
| **IdeaNest用コピー** | `【発言者】` 形式でコピー |

### その他の操作

- **保存**: `.chatnest` ファイルに保存
- **開く**: `.chatnest` ファイルを読み込む
- **全件削除**: ログをすべて削除（確認ダイアログあり）
- 各発言の **✕ボタン**: その発言を個別に削除（確認ダイアログあり）

---

## v0.1.0 実装対象外

以下は v0.1.0 では意図的に実装していません。

- タグ / 検索 / ピン留め / アーカイブ
- タスク管理 / ノート管理 / カード管理
- 複数セッション管理 / 最近使ったファイル
- 自動保存 / スタートダイアログ
- AI 応答 / 外部 API 連携
- クラウド連携 / インターネット接続
- IdeaNest / NoteNest ファイルへの直接書き込み
- Markdown プレビュー / 画像・ファイル添付

「情報を管理する」ツールではなく、「一時的に吐き出してコピーするツール」として設計しています。

## 動作環境

- Windows 10 / 11
- .NET 8 ランタイム

```bash
cd ChatNest
dotnet build
dotnet run --project ChatNest
```
